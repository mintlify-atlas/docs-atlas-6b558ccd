<api method="GET" path="/api/admin/pedidos" />

Retrieve historical orders with optional date and status filtering. Returns up to 200 most recent orders. Admin role required.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token with **Admin** role
</ParamField>

## Query Parameters

<ParamField query="desde" type="datetime" optional>
  Start date filter (UTC). Returns orders created on or after this date.
</ParamField>

<ParamField query="hasta" type="datetime" optional>
  End date filter (UTC). Returns orders created on or before this date.
</ParamField>

<ParamField query="estado" type="enum" optional>
  Filter by order status:
  - `0` - Pendiente
  - `1` - EnPreparacion
  - `2` - Listo
  - `3` - Entregado
  - `4` - Cancelado
</ParamField>

<Note>
  Without filters, returns the 200 most recent orders across all time periods and statuses.
</Note>

## Example Requests

<CodeGroup>

```bash All Recent Orders
curl -X GET https://api.cafeies.local/api/admin/pedidos \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```bash Filter by Date Range
curl -X GET "https://api.cafeies.local/api/admin/pedidos?desde=2024-01-01T00:00:00Z&hasta=2024-01-31T23:59:59Z" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```bash Filter by Status
curl -X GET "https://api.cafeies.local/api/admin/pedidos?estado=3" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```bash Combined Filters
curl -X GET "https://api.cafeies.local/api/admin/pedidos?desde=2024-03-01T00:00:00Z&estado=4" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

</CodeGroup>

## Status Codes

| Code | Description |
|------|-------------|
| 200 | Orders retrieved successfully |
| 401 | Unauthorized |
| 403 | Forbidden (not Admin) |

## Implementation Details

- Returns up to **200 orders** ordered by most recent first
- All timestamps are in UTC
- Includes full order details with line items and user information
- Uses eager loading for optimal performance (includes Lineas, Producto, Usuario)
- Source: `CafeIES.API/Controllers/AdminController.cs:171`
