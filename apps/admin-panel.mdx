## Overview

The CaféIES Admin Panel is a **Blazor WebAssembly** (WASM) application that provides a powerful web interface for cafeteria staff and administrators. It runs entirely in the browser and communicates with the backend API via HTTP.

  <Card title="Complete CRUD" icon="pen-to-square">

  <Card title="Admin-Only Access" icon="shield">

  </Card>

## Architecture

### Blazor WebAssembly Model

Blazor WASM runs .NET code directly in the browser using WebAssembly:

```
Browser
  ├── WebAssembly Runtime (.NET)
  │   ├── CafeIES.Admin.dll
  │   ├── Razor Components (.razor files)
  │   └── Services (AuthAdminService, AdminApiService)
  └── JavaScript Interop (sessionStorage, confirm dialogs)

      ↕ HTTPS

CafeIES.API (ASP.NET Core)
```

### Startup Configuration

```csharp Program.cs
var builder = WebAssemblyHostBuilder.CreateDefault(args);
builder.RootComponents.Add<CafeIES.Admin.App>("#app");
builder.RootComponents.Add<HeadOutlet>("head::after");

// HTTP Client with API base URL
builder.Services.AddScoped(sp => new HttpClient
{
    BaseAddress = new Uri("https://localhost:50658/")
});

// Services
builder.Services.AddScoped<AdminApiService>();
builder.Services.AddScoped<AuthAdminService>();

await builder.Build().RunAsync();
```

<Note>
  Blazor WASM services are registered as **Scoped** (per browser tab) rather than Singleton or Transient.
</Note>

## Authentication

### AuthAdminService

Handles JWT token management and session persistence:

```csharp Services/AuthAdminService.cs
public class AuthAdminService
{
    private readonly HttpClient _http;
    private readonly IJSRuntime _js;

    private string?     _accessToken;
    private string?     _refreshToken;
    private UsuarioDto? _usuario;

    public bool EstaAutenticado => !string.IsNullOrEmpty(_accessToken);
    public string NombreUsuario => _usuario?.NombreCompleto ?? "Admin";

    public async Task<bool> LoginAsync(string email, string password)
    {
        var resp = await _http.PostAsJsonAsync("api/auth/login",
            new LoginRequest(email, password));

        if (!resp.IsSuccessStatusCode) return false;

        var data = await resp.Content.ReadFromJsonAsync<LoginResponse>();
        if (data is null) return false;

        // Only admins can use the panel
        if (data.Usuario.Rol != RolUsuario.Admin) return false;

        _accessToken  = data.AccessToken;
        _refreshToken = data.RefreshToken;
        _usuario      = data.Usuario;

        // Set Authorization header for all future requests
        _http.DefaultRequestHeaders.Authorization =
            new AuthenticationHeaderValue("Bearer", _accessToken);

        // Persist to sessionStorage (survives page refresh)
        await _js.InvokeVoidAsync("sessionStorage.setItem", "admin_token", _accessToken);
        await _js.InvokeVoidAsync("sessionStorage.setItem", "admin_refresh", _refreshToken);

        return true;
    }

    /// <summary>Restore session from sessionStorage (on page reload)</summary>
    public async Task RestaurarSesionAsync()
    {
        _accessToken = await _js.InvokeAsync<string?>("sessionStorage.getItem", "admin_token");
        _refreshToken = await _js.InvokeAsync<string?>("sessionStorage.getItem", "admin_refresh");

        if (!string.IsNullOrEmpty(_accessToken))
            _http.DefaultRequestHeaders.Authorization =
                new AuthenticationHeaderValue("Bearer", _accessToken);
    }

    public async Task LogoutAsync()
    {
        _accessToken  = null;
        _refreshToken = null;
        _usuario      = null;
        _http.DefaultRequestHeaders.Authorization = null;
        await _js.InvokeVoidAsync("sessionStorage.clear");
    }
}
```

<Warning>
  **Session Storage**: Tokens are stored in `sessionStorage`, not `localStorage`. This means they're cleared when the tab closes for better security.
</Warning>

### Login Page

Simple login form that redirects to dashboard on success:

```razor Pages/Login.razor
@page "/login"
@inject AuthAdminService Auth
@inject NavigationManager Nav
@layout EmptyLayout

<div class="login-container">
    <div class="login-card">
        <h1>CaféIES Admin</h1>
        <EditForm Model="@_form" OnValidSubmit="HandleLogin">
            <div class="form-group">
                <label>Email</label>
                <InputText @bind-Value="_form.Email" class="form-input"/>
            </div>
            <div class="form-group">
                <label>Contraseña</label>
                <InputText type="password" @bind-Value="_form.Password" class="form-input"/>
            </div>
            @if (!string.IsNullOrEmpty(_errorMsg))
            {
                <p class="error-msg">@_errorMsg</p>
            }
            <button type="submit" class="btn-primary" disabled="@_loading">
                @(_loading ? "Iniciando sesión..." : "Entrar")
            </button>
        </EditForm>
    </div>
</div>

@code {
    private LoginForm _form = new();
    private bool _loading;
    private string _errorMsg = string.Empty;

    private async Task HandleLogin()
    {
        _loading = true;
        _errorMsg = string.Empty;

        var success = await Auth.LoginAsync(_form.Email, _form.Password);
        _loading = false;

        if (success)
            Nav.NavigateTo("/");
        else
            _errorMsg = "Email o contraseña incorrectos, o no tienes permisos de administrador.";
    }

    class LoginForm
    {
        public string Email { get; set; } = string.Empty;
        public string Password { get; set; } = string.Empty;
    }
}
```

```razor Pages/Dashboard.razor
            }
```

```razor Pages/Usuarios.razor
            {
                <button class="btn-secondary" @onclick="() => Suspender(user.Id)">Suspender</button>
            }
            else if (user.Estado == EstadoCuenta.Suspendida)
            {
                <button class="btn-success" @onclick="() => Reactivar(user.Id)">Reactivar</button>
            }

@code {
    private List<UsuarioDto> _usuarios = new();
    private EstadoCuenta? _filtroEstado = EstadoCuenta.PendienteValidacion;
    private int _pendientesCount;

    protected override async Task OnInitializedAsync()
    {
        await CargarUsuariosAsync();
        _pendientesCount = await Api.GetPendientesCountAsync();
    }

    private async Task CargarUsuariosAsync()
    {
        _usuarios = _filtroEstado.HasValue
            ? await Api.GetUsuariosPendientesAsync()
            : await Api.GetTodosUsuariosAsync();
    }

    private async Task Aprobar(int id)
    {
        await Api.ValidarAlumnoAsync(id, aprobar: true);
        await CargarUsuariosAsync();
    }

    private async Task Rechazar(int id)
    {
        await Api.ValidarAlumnoAsync(id, aprobar: false);
        await CargarUsuariosAsync();
    }
}
```

### AdminApiService

Centralized HTTP client for all admin operations:

```csharp Services/AdminApiService.cs
public class AdminApiService
{
    private readonly HttpClient _http;

    public AdminApiService(HttpClient http)
    {
        _http = http;
    }

    // Dashboard
    public async Task<DashboardDto?> GetDashboardAsync()
    {
        try { return await _http.GetFromJsonAsync<DashboardDto>("api/admin/dashboard"); }
        catch { return null; }
    }

    // Products
    public async Task<List<ProductoDto>> GetProductosAsync()
    {
        try { return await _http.GetFromJsonAsync<List<ProductoDto>>("api/productos?soloActivos=false") ?? new(); }
        catch { return new(); }
    }

    public async Task<bool> CrearProductoAsync(CrearProductoRequest req)
    {
        try { return (await _http.PostAsJsonAsync("api/productos", req)).IsSuccessStatusCode; }
        catch { return false; }
    }

    public async Task<bool> ActualizarStockAsync(int id, int nuevoStock)
    {
        try { return (await _http.PatchAsJsonAsync($"api/productos/{id}/stock", new ActualizarStockRequest(nuevoStock))).IsSuccessStatusCode; }
        catch { return false; }
    }

    // Users
    public async Task<List<UsuarioDto>> GetTodosUsuariosAsync()
    {
        try { return await _http.GetFromJsonAsync<List<UsuarioDto>>("api/admin/usuarios") ?? new(); }
        catch { return new(); }
    }

    public async Task<bool> ValidarAlumnoAsync(int id, bool aprobar)
    {
        try { return (await _http.PatchAsync($"api/admin/usuarios/{id}/validar?aprobar={aprobar}", null)).IsSuccessStatusCode; }
        catch { return false; }
    }

    // Orders
    public async Task<bool> CambiarEstadoPedidoAsync(int id, EstadoPedido estado)
    {
        try { return (await _http.PatchAsJsonAsync($"api/pedidos/{id}/estado", new CambiarEstadoRequest(estado))).IsSuccessStatusCode; }
        catch { return false; }
    }

    // Categories, Schedules, Invitations...
}
```

### CORS Configuration

The API must allow the admin panel origin:

```csharp Program.cs (API)
builder.Services.AddCors(opt =>
    opt.AddPolicy("AllowAdmin", p =>
        p.SetIsOriginAllowed(origin =>
                origin.StartsWith("https://localhost") ||
                origin.StartsWith("http://localhost") ||

         .AllowAnyMethod()
```

## Performance Considerations

<CardGroup cols={2}>
  <Card title="Lazy Loading" icon="clock">
    Blazor WASM downloads assemblies on demand. Initial load can be slow, but subsequent navigation is instant.
  </Card>
  <Card title="Caching" icon="database">
    Browser caches assemblies. Use cache-busting for deployments.
  </Card>
  <Card title="Pagination" icon="list">
    Large data sets should be paginated server-side to avoid downloading thousands of records.
  </Card>
  <Card title="Compression" icon="file-zipper">
    Enable Brotli compression on your web server for smaller download sizes.
  </Card>
</CardGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="White screen / blank page">
    Check browser console for errors. Common causes:
    - API not running or wrong base URL
    - CORS policy blocking requests
    - JavaScript errors preventing render
  </Accordion>
  <Accordion title="Authentication loop (keeps redirecting to login)">
    Session storage may be cleared. Check that `RestaurarSesionAsync()` is called on app startup.
  </Accordion>
  <Accordion title="API calls return 401 Unauthorized">
    JWT token expired or invalid. Implement token refresh logic in `AuthAdminService`.
  </Accordion>
  <Accordion title="Slow initial load">
    Blazor WASM downloads .NET runtime and assemblies. This is normal. Use a loading screen and consider AOT compilation for production.
  </Accordion>
</AccordionGroup>
