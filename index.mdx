## Use Cases

<AccordionGroup>
  <Accordion title="For Students">
    Browse the menu, place orders during allowed time windows, track order status in real-time, and pick up when ready. All from your mobile device.
  </Accordion>
  <Accordion title="For Teachers & Staff">
    No time restrictions — order anytime during operating hours. Register using QR codes provided by administrators.
  </Accordion>
  <Accordion title="For Administrators">
    Manage products, validate student registrations, generate staff invitations, configure time schedules, and monitor all orders through the admin panel.
  </Accordion>
  <Accordion title="For Cafeteria Staff">
    Receive orders instantly, update preparation status, manage stock levels, and communicate with customers through order states.
  </Accordion>

## Technology Stack

Built with modern .NET technologies for performance, scalability, and maintainability:

- **Backend**: ASP.NET Core 9 with Entity Framework Core
- **Mobile**: .NET MAUI for cross-platform iOS/Android
- **Admin**: Blazor WebAssembly for responsive web interface
- **Database**: SQL Server with EF Core migrations
- **Real-Time**: SignalR for live order updates
- **Authentication**: JWT with refresh tokens and BCrypt password hashing
